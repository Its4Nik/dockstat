# Themes

To set the default theme please use the Name provided by DaisyUI.

```
daisyui: {
    themes: ["light", "nord", "dracula", "sunset", "night", "black", "valentine", "forest", "business"],
  },
```

ℹ️ "valentine" is known as "Pastel"

ℹ️ "black" is known as "Amoled"
